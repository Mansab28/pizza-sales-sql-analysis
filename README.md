# 🍕 Pizza Sales Analysis Using SQL Server

## Project Overview

This project analyzes a pizza sales dataset using SQL Server. The dataset was imported from a CSV file into SQL Server, and SQL queries were used to answer key business questions related to sales performance and customer ordering behavior.

## Tools Used

- SQL Server
- SQL
- CSV Dataset

## Business Questions Answered

- Total Revenue
- Average Order Value
- Total Pizzas Sold
- Total Orders Placed
- Average Pizzas Per Order
- Daily Sales Trend
- Hourly Sales Trend
- Sales Percentage by Pizza Category
- Sales Percentage by Pizza Size
- Units Sold per Pizza Category
- Top 5 Best-Selling Pizzas
- Top 5 Underperforming Pizzas

## SQL Concepts Used

- SELECT
- WHERE
- GROUP BY
- ORDER BY
- Aggregate Functions
- DISTINCT
- CAST
- DATEPART
- DATENAME
- Subqueries

## Project Files
- 'pizza_sales_dataset.csv'  - Dataset Used
- 'Pizza_sales_SQL_Project.pdf' – Project report containing SQL queries, screenshots, and query results.
